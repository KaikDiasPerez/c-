# **Aula de TPA**
## **Markdown**
### **Markdown** é uma linguagem de marcação leve com uma sintaxe simples usada para formatar texto. Ela permite que as pessoas escrevam usando texto simples e, com a ajuda de caracteres de formatação (como: *, #, _), transformem esse texto em um formato mais elaborado, como HTML, sem a complexidade de escrever o código diretamente. É frequentemente utilizada para criar arquivos de documentação (como README.md), escrever posts em blogs, criar notas e formatar mensagens em plataformas como o Discord.
